¶ Creates a tag

Creates a tag if it doesn't exist, during the maintenance call.

  1. Checks if the organization ID is present. If organization is not present, does no further processing.
  2. Check if the tag belongs to the organization. If the tag exists, or belongs to another organization an error message is sent to the caller.
  3. Creates a new tag.
  4. Creates a tag info pending event.

serviceObj serviceObj containing the tag maintenance data
